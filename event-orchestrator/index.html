Event Orchestrator

Plan and manage events effortlessly — guests, budgets, vendors, floor plans, and more. Native on iPhone, iPad, and Mac, powered by on-device AI.

Coming to the App Store

Requires iOS / iPadOS 26+  ·  macOS 26+

Everything you need to plan an event

From the first idea to the day-of schedule, Event Orchestrator keeps every detail in one place.

Events

Create and manage events with cover photos, date ranges, venue details, and status tracking.

Tasks

Checklists with templates, progress tracking, and status-aware swipe actions to stay on top of every to-do.

Guests

Contact import with deduplication, dietary restrictions, and RSVP tracking for every attendee.

Budget

Line items with over-budget warnings, a donut chart summary, and vendor offering import.

Vendors

Global vendor directory with per-event linking, AI quote parsing, and service offerings management.

Floor Plan

Canvas editor with AI-generated layout suggestions and an edge clearance system for precise placement.

Schedule

Day-of timeline blocks so every moment of your event runs exactly as planned.

Notes

Rich text notes with attachments, synced across devices via CloudKit.

Widgets & PDF Export

Home and lock screen widgets for countdowns and tasks. Export guest lists, budgets, and more as PDF.

See it in action

A clean, native interface designed for iPhone, iPad, and Mac.

Powered by on-device AI

Event Orchestrator uses Apple's on-device Foundation Models — your data never leaves your device. AI assists with floor plan layouts, vendor quote parsing, and more, all privately and instantly.
